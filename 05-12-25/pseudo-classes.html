Pseudo-Classes

Uma pseudo-classe é usada para definir um estado especial de um elemento.

Por exemplo:

  • Estilizar um elemento quando um usuário passar o mouse sobre ele
  • Links visitados e não visitados de forma diferente
  • Estilizar um elemento quando ele é focado

As pseudo-classes são definidas com dois pontos (:) seguidos do nome da pseudo-classe.

Exemplo:


    p:hover {
        color: red;
    }

a:link{propriedade:valor} (em link não visitado)

a:visited{propriedade:valor} (em link visitado)

a:active{propriedade:valor} (em link já selecionado pelo usuário)

a:hover{propriedade:valor} (quando o mouse estiver sobre o elemento)

Exemplo:


    a:link {
        color: green;
    }
    a:visited {
        color: purple;
    }
    a:active {
        color: red;
    }
    a:hover {
        color: red;
    }
Referencia

a:first-child{propriedade:valor} (primeiro objeto de uma classe)

a:last-child{propriedade:valor} (ultimo objeto de uma classe)

a:nth-child(n){propriedade:valor} (n é um número que representa a posição do objeto, pode ser par ou impar tambem)

Exemplo:


    p:first-child {
        color: red;
    }

    p:last-child {
        color: blue;
    }
    p:nth-child(2) {
        color: green;
    }
