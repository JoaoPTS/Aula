Pseudo-Elementos

Um pseudo-elemento é usado para definir um estilo especial para uma parte de um elemento.

Por exemplo:

  • Estilizar a primeira linha de um elemento
  • Estilizar o primeiro caractere de um elemento
  • Adicionar conteúdo antes ou depois de um elemento

Os pseudo-elementos são definidos com dois pontos (:) seguidos do nome do pseudo-elemento.

Exemplo:

p::first-line{propriedade:valor} (primeira linha de um elemento)

p::first-letter{propriedade:valor} (primeira letra de um elemento)

p::before{propriedade:valor} (antes do elemento)

p::after{propriedade:valor} (depois do elemento)

::selection{propriedade:valor} (seleciona o texto)

::marker{propriedade:valor} (estiliza o marcador de uma lista)


    p::first-line {
        color: red;
        }
    p::first-letter {
        color: blue;
        font-size: 2em;
    }
    p::before {
        content: "Antes do parágrafo. ";
        color: green;
    }
    p::after {
        content: "Depois do parágrafo.";
        color: green;
    }
    ::selection {
        background: yellow;
        color: red;
    }
    ::marker {
        color: red;
        font-size: 2em;
    }
