Float

O float é uma propriedade do CSS que permite que um elemento flutue para a esquerda ou para a direita dentro de seu contêiner pai. Isso significa que o elemento pode ser posicionado em um lado do contêiner, permitindo que outros elementos se ajustem ao seu redor.

Utilizando

Float esquerda

float: left;

Float direita

float: right;

Float nenhum

float: none;
